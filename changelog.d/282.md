- Command mode (M4, issue #282, ac7-p0): the local model no longer pastes a
  narrated prompt over your selection. A blank or single-stray-character
  transcribed instruction now short-circuits to a clear "no instruction
  heard" notice before the model is ever called, and any transform result
  that looks like a conversational preamble or a paraphrase of the system
  prompt (rather than the rewritten selection) is rejected — the clipboard is
  restored and nothing is pasted. Also hardens the command prompt against
  preambles via a new versioned `command_v2.txt` (anti-narration rule plus a
  worked few-shot example), leaving `command_v1.txt` untouched.
