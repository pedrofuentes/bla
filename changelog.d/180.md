- File-mode output-path/template picker (issue #180, AC-7 #173 p0): the settings window's General tab now has an Output section — a Paste-at-cursor/Append-to-a-file mode switch, plus, when File mode is selected, a "Base folder (vault)" field and a `{{date:YYYY-MM-DD}}`-templated path field (e.g. `daily/{{date:YYYY-MM-DD}}.md` for an Obsidian daily note). Switching to File mode now actually writes into the folder you choose instead of a fixed, unchangeable default location; an invalid path template (absolute, or one that escapes the base folder) shows an inline error and is never saved.
