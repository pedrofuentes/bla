- Cleanup / per-app tone (M4, issue #283, ac7-p0): dictation no longer emits
  a conversational preamble as the cleaned text (e.g. "This is a formal
  rewrite of your original transcript: …"). When the local model returns a
  preamble/label instead of only the rewritten transcript, the pipeline now
  falls back to the deterministic regex baseline — the same safe degradation
  already used when the model is unreachable — so the label never reaches the
  output. Also hardens the cleanup/tone prompt via a new versioned
  `cleanup_v4.txt` (anti-preamble rule plus a worked few-shot example),
  leaving `cleanup_v3.txt` untouched.
